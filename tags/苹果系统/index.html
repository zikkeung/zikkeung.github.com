MAC OSX 配置命令

备忘自己常用到的命令。

开关隐藏文件

把 TRUE 换曾 FALSE 就可以关闭显示隐藏文件。

在Finder标题栏显示当前路径

Go2Shell支持ITerm2

Go2Shell默认打开系统自带的终端，可通过下面的命令调出配置框，选择iTern2或xtern。

开关time machina本地快照

time machina 会占用磁盘来作备份空间，等插入备份盘就会将这些备份转移过去，如果不想占空磁盘空间，可将本地快照关闭。

开关Dashboard

Dashboard鸡肋的功能，建议关闭。

YES换成NO就可以恢复
